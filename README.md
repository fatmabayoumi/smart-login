# Smart Login System

A complete login/signup system with HTML, CSS, and pure JavaScript.

## Features
- User registration with success message
- Login authentication
- Welcome page with personalized greeting
- Local storage for user data
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Pure)

## How to Use
1. Sign up with your name, email, and password
2. Login with your credentials
3. View your personalized welcome page
